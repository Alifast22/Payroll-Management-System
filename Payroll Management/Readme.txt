How to run frontend?

1)Use command npx create-react-app my-app on vs code terminal(frontend dependencies and basic node modules will be installed
2)Add the src folder with the given src folder in frontend folder
3)Replace Package and Package-lock files
4)move to src folder through cd and type npm start


How to run backend?

1) npm init -y

"Command for initializing Node packages"

2) npm add express dotenv multer

-)express(express server)
-)dotenv(for environment variables)
-)multer(for images)

3) npm i nodemon

"Command for installing auto save and reload server"

4)
Replace Package,Package-lock and Routes folder
 
5) For running server:

move to the folder where app.js is placed and write nodemon app.js
